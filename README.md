# 💰 Next.js Financial Dashboard

A full-stack financial dashboard built with **Next.js**, featuring authentication, invoice management, protected routes, and a PostgreSQL database integration. This project is designed as a hands-on learning application that demonstrates modern Next.js development practices.

## ✨ Features

- 🌐 Public landing page
- 🔐 User authentication and protected dashboard
- 📊 Dashboard overview
- 🧾 Create, edit, and delete invoices
- 🔍 Search and pagination
- 🗄️ PostgreSQL database integration
- ⚡ Server Actions for data mutations
- 🚀 Optimized images, fonts, and routing
- ❌ Error handling (404 & application errors)
- ✅ Server-side form validation
- ♿ Accessibility best practices
- 📱 SEO and metadata support

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm, pnpm, or yarn
- PostgreSQL database

### Installation

```bash
git clone https://github.com/your-username/financial-dashboard.git

cd financial-dashboard

npm install
```

Create a `.env.local` file and configure the required environment variables.

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
